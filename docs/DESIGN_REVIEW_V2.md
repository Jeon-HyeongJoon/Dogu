# 디자인 평가 & v2 리디자인 방향 (2026-07)

## 1. 현재 디자인 평가

### v1 (기본 앱, `/`)

모노크롬(화이트 `#ffffff` + 잉크 `#0a0a0a`) 위에 딥 그린(`#13402d`) 액센트, 브랜드
타이틀은 HSBombaram 손글씨체. 헤더는 로고+타이틀 락업, 5탭 하단바.

| 항목 | 평가 |
|---|---|
| 톤 | 차분하고 부드러움 — "깔끔"은 충족, "공격적"과는 거리가 멂 |
| 타이포 | 본문 Pretendard는 안정적이나 디스플레이 위계(크기 점프)가 약해 임팩트 부족 |
| 상품 카드 | 여백·헤어라인 위주라 단정하지만 가격/할인 강조가 약해 구매 유도력이 낮음 |
| CTA | 그린 계열 버튼 — 화면 안에서 시선을 끌지 못함 |

강점: 정보 구조(5탭, 섹션 구성)와 기능 배선(store 공유)이 탄탄하다. 약점: 판매
압력(할인·가격·행동 유도)이 시각적으로 드러나지 않는다.

### v2 현행 (유희왕 "욕망의 항아리" 테마, `/v2`)

파치먼트 배경 + 청록 카드 프레임 + 금색 트림 + 크림 패널. 모든 타일이 "마법 카드"
프레임 구조.

| 항목 | 평가 |
|---|---|
| 개성 | 테마성은 강하지만 쇼핑몰이 아니라 TCG 컬렉션 앱으로 읽힘 |
| 노이즈 | 타일마다 2중 테두리(청록+금)·크림 박스가 겹쳐 시각 소음이 큼 — "깔끔" 실패 |
| 상품 전달 | 아트워크(이니셜 글리프)가 프레임 장식에 눌려 상품 자체가 주인공이 아님 |
| 카피 | "속공 마법", "드로우", "카드 상세" 등 은유가 커머스 신뢰도를 깎음 |
| 대비 | 파치먼트 위 갈색 잉크 — 대비가 낮아 가격·할인의 펀치력이 없음 |

## 2. v2 리디자인 방향 — "Aggressive & Clean"

목표: 무신사·29CM 계열의 고대비 커머스 문법. **공격적** = 크고 무거운 타이포,
핫 액센트 하나로 가격·할인·CTA를 때린다. **깔끔** = 순백 배경, 헤어라인 구분선,
장식(프레임·트림·패널) 전부 제거.

### 토큰

| 토큰 | 값 | 용도 |
|---|---|---|
| `bg` | `#ffffff` | 페이지 배경 (순백) |
| `surface` | `#f4f4f2` | 서브 서피스(칩·입력창) |
| `ink` | `#0f0f0f` | 본문·디스플레이 잉크 (거의 검정) |
| `inkSoft` / `inkFaint` | `#5c5c5c` / `#9c9c9c` | 보조 텍스트 |
| `line` | `#e7e7e5` | 헤어라인 |
| `jet` | `#111111` | 헤더·히어로·CTA 블록 (블랙 블록) |
| `jetInk` | `#f5f5f3` | 블랙 블록 위 글씨 |
| `accent` | `#ff3d00` | 가격·할인·활성 상태 — 유일한 유채색 |

### 원칙

1. **블랙 블록으로 때린다** — 헤더·히어로·주요 CTA는 제트 블랙. 나머지는 순백.
2. **액센트는 하나** — `#ff3d00`은 가격/할인/활성 표시에만. 두 번째 유채색 금지.
3. **타이포가 레이아웃이다** — 디스플레이는 Pretendard ExtraBold, 자간 −1,
   섹션 타이틀 22–24. 잉글리시 레이블은 대문자+자간 확대(eyebrow).
4. **장식 제로** — 카드 프레임·이중 테두리·그림자·크림 패널 제거. 구분은 여백과
   헤어라인만. 라운드는 0–4.
5. **상품이 주인공** — 아트워크 풀블리드(테두리 없음), 할인율은 액센트 대형 표기,
   가격은 타이틀보다 크게.
6. **카피는 직설** — TCG 은유 제거. "오늘의 특가", "신상품", "상품 상세".
